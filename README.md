# simple-ftp
CSCI 4780 Project 1 Implements a basic FTP client-server program.

### Compile Client:
```
$ javac myftp.java
```
### Compile Server:
```
$ javac myftpserver.java
```

### Run Client:
```
$ java myftp [MACHINE_NAME] [PORT]
```

### Run Server:
```
$ java myftpserver [PORT]
```

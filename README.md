# simple-ftp project
Josh Messitte, Alex Holmes, Robert Urquhart

### Compile Client:
```
$ javac SimpleFTP.java
```
### Compile Server:
```
$ javac SimpleFTPServer.java
```

### Run Client:
```
$ java SimpleFTP [MACHINE_NAME] [nPORT] [tPORT]
```

### Run Server:
```
$ java SimpleFTPServer [nPORT] [tPORT]
```
